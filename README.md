# ChainTools GitLab

Ce projet à pour but de monter une chain tools, celle-ci comprend : 

- Un serveur barre métal en Debian 10
- Des VM générés à l'aide de Vagrant et Ansible
- Créer une VM en Debian/Jessie qui suportera un serveur GitLab (toujours à l'aide de Vagrant et Ansible) 
- Créer une VM qui servira de serveur DNS
